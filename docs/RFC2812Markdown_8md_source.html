19 ### 2. Command-Reply Sequences (Mandatory part)
20 
21 >The server reply is defined to contain the 3-digit code, followed by space
22 > **< SP >** , followed by one line of text, and terminated by the Irc end-of-line code **< CRLF >** (aka ```\r\n```).
